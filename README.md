# Code Editor Website

A full-stack online Code Editor web application developed using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript. The application allows users to write, compile, and execute code online using the Judge0 API with secure user authentication.

## Features

- User Authentication (Login & Signup)
- Online Code Editor
- Execute Code using Judge0 API
- Real-time Output Display
- Multiple Programming Language Support
- Responsive User Interface
- MongoDB Database Integration
- Secure Password Encryption
- JWT-based Authentication

## Installation

Clone the repository:

```bash
git clone https://github.com/nandini281105/code-editor-website.git
```

Navigate to the project folder:

```bash
cd code-editor-website
```

Install dependencies:

```bash
npm install
```

Configure MongoDB by adding your MongoDB connection string.

Start the backend server:

```bash
npm start
```

or

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3001
```
